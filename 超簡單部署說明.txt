🎯 超級簡單部署說明 - 只需要做這些事：

第一步：取得 GitHub Token
1. 開啟網頁：https://github.com
2. 登入您的帳號
3. 點擊右上角您的頭像
4. 點擊 Settings (設定)
5. 在左邊找到 Developer settings (開發者設定)
6. 點擊 Personal access tokens (個人存取權杖)
7. 點擊 Generate new token (產生新的權杖)
8. 勾選這些權限：
   ✓ repo (這個很重要)
   ✓ workflow (工作流程)
9. 點擊 Generate token (產生權杖)
10. 複製出現的英文字串 (只會出現一次！)

第二步：執行部署
1. 雙擊檔案：一鍵部署.bat
2. 貼上您剛才複製的 Token
3. 按 Enter 鍵
4. 等待完成

第三步：完成
- 程式會告訴您網址
- 幾分鐘後就能看到您的網站

🆘 遇到問題：
- Token 複製錯誤：重新產生一個
- 網頁找不到：等待 5 分鐘再試
- 其他問題：把錯誤訊息告訴我

💾 記住：
- Token 就像密碼，要好好保存
- 以後修改網站，程式會自動更新
- 完全免費！
