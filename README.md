# 蔡紘的个人网站

一个现代化、美观的个人作品集网站，展示专业技能和项目经验。

## ✨ 特性

- 🎨 **现代化设计** - 使用 TailwindCSS 打造的优雅界面
- 📱 **完全响应式** - 在所有设备上都能完美显示
- ⚡ **流畅动画** - 精心设计的交互动效和过渡效果
- 🌙 **深色模式支持** - 舒适的浏览体验
- 🔍 **SEO 优化** - 良好的搜索引擎优化
- 📧 **联系表单** - 便捷的联系和沟通方式

## 🛠️ 技术栈

- **HTML5** - 语义化标记
- **TailwindCSS** - 实用优先的 CSS 框架
- **JavaScript (ES6+)** - 现代交互功能
- **Font Awesome** - 图标库
- **Google Fonts** - 优质字体

## 📁 项目结构

```
personal-website/
├── index.html          # 主页面
├── script.js           # JavaScript 功能
├── README.md           # 项目文档
└── assets/             # 静态资源（如果需要）
```

## 🚀 快速开始

1. 克隆或下载项目文件
2. 在浏览器中打开 `index.html`
3. 或使用本地服务器：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve .

# 使用 PHP
php -S localhost:8000
```

4. 访问 `http://localhost:8000`

## 📝 自定义内容

### 修改个人信息

在 `index.html` 中找到以下部分并修改：

- **姓名**: 修改 `<h1>` 标签中的 "蔡紘"
- **职业描述**: 修改 `script.js` 中的 `roles` 数组
- **关于我**: 修改 "关于" 部分的文本内容
- **联系方式**: 修改联系部分的邮箱、电话等信息

### 修改技能和项目

在相应的 HTML 部分修改：

- **技能**: 在 "技能专长" 部分添加或修改技能类别
- **项目**: 在 "项目展示" 部分添加或修改项目信息

### 自定义样式

主要样式使用 TailwindCSS，可以通过修改以下部分来自定义：

- **渐变色**: 修改 `.gradient-text` 和 `.hero-gradient` 的 CSS
- **颜色主题**: 修改 TailwindCSS 类名中的颜色值
- **字体**: 修改 Google Fonts 的字体选择
